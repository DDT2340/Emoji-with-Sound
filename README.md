## üöÄ What is Soundji?

**Soundji = Emoji + Sound.**  
This project adds sound effects to emoji reactions. When a user sends an emoji in a chat, the corresponding sound plays on the recipient's device.

### üîä Example Use Cases:
- üòÇ Laugh emoji plays a giggle sound  
- üîî Bell emoji triggers a chime  
- üéâ Celebration emoji plays a party horn
